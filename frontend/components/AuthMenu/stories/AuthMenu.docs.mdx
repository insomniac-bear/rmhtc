# AuthMenu

```tsx
import { AuthMenu } from '../{Relative path}/AuthMenu';
```
## Компонент "Меню авторизации и регистарции". 

  Меню с двумя кнопками. Кнопки `Sign Up` и `Log In` откарывают модальные окна для регистрации и авторизации пользователя, соответсвенно. 
  Компонент используется в шапке сайта.
 
```tsx
  <AuthMenu />
```


### Props
| Свойство                      | Тип                                     | По умолчанию | Описание                                                      |
|-------------------------------|-----------------------------------------|--------------|---------------------------------------------------------------|
| className                     | 'string'                                |       -      | Дополнительный, внешний css-класс                             |
| ...props                      | -                                       |       -      | Внешние атрибуты                                              |




### Структура компонента
```tsx
      export const AuthMenu: FC<IAuthMenuProps> = ({ className = '', ...props }) => (
        <ul className={`${styles.authMenu} ${className}`} {...props}>
          <li>
            <Link href="/?modal=signup" passHref>
              <a>
                <Button appearance="primary" type="button">Sign Up</Button>
              </a>
            </Link>
          </li>
          <li>
            <Link href="/?modal=login" passHref>
              <a>
                <Button appearance="ghost" type="button">Log In</Button>
              </a>
            </Link>
          </li>
        </ul>
      );
```
