# CatalogCompanyPreview

```tsx
import { CatalogCompanyPreview } from '../{Relative path}/CatalogCompanyPreview';
```
## Компонент "Превью компании в каталоге". 
  Карточка превью компании. Принимает параметром полный объект компании. Используется в каталоге компаний, отображает основную информацию о компании. 

```tsx
  <CatalogCompanyPreview company={companyData} />
```

### Props
| Свойство                      | Тип                                     | По умолчанию | Описание                                                      |
|-------------------------------|-----------------------------------------|--------------|---------------------------------------------------------------|
| company                       | 'object'                                |       -      | Полный объект компании                                        |
| className                     | 'string'                                |       -      | Дополнительный, внешний css-класс                             |
| ...props                      | -                                       |       -      | Внешние атрибуты                                              |
