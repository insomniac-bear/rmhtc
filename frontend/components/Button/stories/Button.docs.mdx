## Компонент "Кнопка". 

Кнопка. Используется, когда человеку нужно сделать выбор, подтвердить или совершить какое-то действие. Например, отправить сообщение, оплатить заказ, сохранить изменения, войти в аккаунт.

Кнопка в фокусе визуально выделяется, чтобы человек понимал, что вот-вот нажмёт на неё и что-то произойдёт.

Текст на кнопке показывает, за какое действие она отвечает. Сделайте его однозначным, понятным и коротким.

### Внеший вид `appearance`
  Имеется два варианта: 
  - С полной заливкой заднего фона - `primary`
  - С границей - `ghost` 

```tsx
  <Button
    appearance="primary"
    onClick={() => {}}
    type="reset"
  >
    Click me
  </Button>
```



### Props
| Свойство                      | Тип                                     | По умолчанию | Описание                                                      |
|-------------------------------|-----------------------------------------|--------------|---------------------------------------------------------------|
| type                          | "button" , "submit" , "reset"           |       -      | Отвечает за выбор Html-типа кнопки                            |
| appearance                    | "ghost" , "primary"                     |       -      | Вариант внешнего вида                                         |
| disabled                      | "boolean"                               |       -      | Состояние блокировки кноки                                    |
| children                      | -                                       |       -      | Текст или элемент, обернутый в кнопку                         |
| onClick                       | 'function'                              |       -      | Функция - обработчик события `click`                          |
| className                     | 'string'                                |       -      | Дополнительный, внешний css-класс                             |
| ...props                      | -                                       |       -      | Внешние атрибуты                                              |




### Структура компонента
```tsx
      export const Button: FC<IButtonProps> = ({
        type = 'button', className = '', onClick, children, appearance = 'ghost', disabled, ...props
      }) => (
        <button
          className={`
          ${styles.button}
          ${appearance ? styles[`button_appearance_${appearance}`] : ''}
          ${className}
          `}
          type={type}
          onClick={onClick}
          disabled={disabled}
          {...props}
        >
          {children}
        </button>
      );
```
